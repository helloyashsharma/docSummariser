
# docSummariser

Python app to summarise docs using chatgpt.


## Run Locally

Clone the project

```bash
  git clone https://github.com/helloyashsharma/docSummariser.git
```

Go to the project directory

```bash
  cd docSummariser
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Update .env file
```bash
OPENAI_API_KEY=
LEGAL_ASSISTANT_ID=
```

Run the project

```bash
  cd src
  python ui.py
```
